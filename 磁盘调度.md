# 选择题

1. 对磁盘进行移臂调度的目的是为了缩短(     A )时间。
    A 寻找            B 延迟             C 传送                D 启动

1. 下列算法中用于磁盘移臂调度的是(　C　)
A  时间片轮转法　　B  LRU算法　　
C  最短寻找时间优先算法　　D  优先级高者优先算法　　

1. 一个进程的读磁盘操作完成后，操作系统针对该进程必做的是（A）。   
A.修改进程状态为就绪态                 B.降低进程优先级   
C.进程分配用户内存空间                 D.增加进程的时间片大小 
【解析】进程等待I/O操作完成便会从阻塞状态转换到就绪状态。
